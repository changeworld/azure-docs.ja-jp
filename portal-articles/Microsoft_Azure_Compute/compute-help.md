# <a name="virtual-machines-documentation"></a>Virtual Machines のドキュメント
Azure Virtual Machines では、セキュリティで保護された高スケールな仮想化インフラストラクチャをオンデマンドで構築できます。 Windows 仮想マシンを作成できるほか、Red Hat や Ubuntu など、お好みの Linux ディストリビューションを実行する Linux 仮想マシンを作成できます。 その方法については、次を参照してください。

- [Windows 仮想マシンの作成](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot)
- [Linux 仮想マシンの作成](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot)

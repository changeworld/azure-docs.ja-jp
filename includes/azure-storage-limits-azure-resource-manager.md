| リソース | 制限 |
| --- | --- |
| Storage アカウント管理操作数 (読み取り) |5 分あたり 800 |
| Storage アカウント管理操作数 (書き込み) |1 秒あたり 10 または 1 時間あたり 1,200 |
| Storage アカウント管理操作数 (リスト) |5 分あたり 100 |

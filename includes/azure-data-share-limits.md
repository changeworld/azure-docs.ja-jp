組織では、Azure Data Share を利用することで、顧客やパートナーとデータを簡単かつ安全に共有することができます。

| **リソース** | **制限** |
| --- | --- |
| Azure サブスクリプションごとのデータ共有リソースの最大数 |50 |
| データ共有リソースごとの送信された共有の最大数 |100 |
| データ共有リソースごとの受信された共有の最大数 |100 |
| 送信された共有ごとの招待の最大数 |100 |
| 送信された共有ごとの共有サブスクリプションの最大数 |100 |
| 共有ごとのデータセットの最大数 |100 |
| 共有ごとのスナップショット スケジュールの最大数 |1 |

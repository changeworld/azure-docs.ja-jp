使用するアクティブ化キーは...

- Azure Active Directory (Azure AD) ベースの認証キーです。
- 3 日で有効期限が切れます。
- デバイスがアクティブ化された後は使用されません。

デバイスをアクティブ化すると、そのデバイスはトークンを使用して Azure と通信します。

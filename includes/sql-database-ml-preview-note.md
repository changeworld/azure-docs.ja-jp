> Azure SQL Database の Machine Learning service は、現在プレビュー期間です。 プレビューへのサインアップは、現在 **終了** しています。

> Azure SQL での機械学習の使用を続行するには、「[**Azure SQL Managed Instance の Machine Learning Services** (プレビュー)](../articles/azure-sql/managed-instance/machine-learning-services-overview.md)」を参照してください。

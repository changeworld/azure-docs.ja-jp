# <a name="customize-your-azure-security-center-for-iot-solution"></a>Azure Security Center for IoT ソリューションをカスタマイズする 

このガイドでは、Azure Security Center for IoT のさまざまな設定をカスタマイズする方法について説明します。  

> [!div class="checklist"]
> * ソリューションの推奨事項を構成する 
> * 設定を変更する 

## <a name="change-settings"></a>設定を変更する

Azure Security Center for IoT の設定を管理します。

IoT Hub 上で、左上隅のセキュリティの概要ブレードに移動し、[設定] を確認します。Azure Security Center for IoT の設定を構成するには、以下の手順を実行します。

1. Azure portal で **[IoT Hub]** を開きます。 
1. 左側のメニューの **[セキュリティ]** の下の **[概要]** を選択して開きます。
1. [設定] の下で、変更するソリューション設定を選択します。   
1. どの設定画面でも、設定の変更を保存するために、上部の **[保存]** を常にクリックしてください。 

## <a name="configure-solution-recommendations"></a>ソリューションの推奨事項を構成する

Azure Security Center for IoT ソリューションの推奨事項を構成するには、以下の手順を実行します。

1. Azure portal で **[IoT Hub]** を開きます。 
1. 左側のメニューの **[セキュリティ]** の下の **[概要]** を選択して開きます。
1. 左側のメニューで、[設定] の下の **[Recommended Configuration]\(推奨される構成\)** を選択します。 
1. 組織およびワークフローに関連するソリューションの推奨事項を無効にするか、有効にします。 
1. 画面の上部にある **[保存]** を選択して選択内容を保存します。

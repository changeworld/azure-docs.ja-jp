# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory のホワイトペーパー 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

ホワイトペーパーでは、Azure Data Factory の概念がより深いレベルで説明されています。 この記事では、Azure Data Factory で利用可能なホワイト ペーパーの一覧を示します。

| **ホワイトペーパー** | **説明** |
| --- | --- |
|[Azure Data Factory - クラウドでのデータ統合](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | このホワイト ペーパーでは、最新のデータ ウェアハウスを構築、インテリジェントな SaaS アプリケーションを推進し、SQL Server Integration Services パッケージを Azure に高めるためのAzure Data Factory の使用方法について説明します。|
|[Azure Data Factory を使用してオンプレミスのリレーショナル データ ウェアハウスから Azure にデータを移行する](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | このホワイト ペーパーでは、数十テラバイトものデータ既存のオンプレミス リレーショナル データ ウェアハウス (Netezza、Oracle、Teradata、SQL サーバー等) から Azure (Blob Storage or Azure Data Lake Storage 等) へ Azure Data Factory を使用して移行する際の複雑さについて説明しています。 Azure Data Factory でビッグ データを Azure へ注入するという挑戦は、回復性、パフォーマンス、スケーラビリティ、管理、およびセキュリティの点からベスト プラクティスが示されます。 |
|[Azure Data Factory を使用した SAP データ統合](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | このホワイトペーパーでは、SAP データ統合に関して現在サポートされている Azure Data Factory の概要を示します。これには、ターゲット シナリオ、さまざまな要件の SAP コネクタ オプションと比較、Data Factory の各 SAP コネクタの概要が含まれます。 | 
|[Azure Data Factory:クラウドでの SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| このぺーパーは、既存の SSIS ワークロードを Azure Data Factory に移行し、共通の懸案事項に対処する理由について述べています。 次に、Azure-SSIS IR を作成する際の技術的な情報を紹介し、おそらく SQL Server Management Studio (SSMS) など馴染みのあるツールで Azure Data Factory を介して自社パッケージをアップロード、実行および監視する方法を示します。 |
|[Azure data Factory –Passing Parameters (Azure Data Factory - パラメーターを渡す)](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| この記事では、パイプラインとアクティビティ間、およびアクティビティ間でパラメーターを渡す方法について説明します。 |
|[Azure Data Factory - DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| このホワイト ペーパーでは、Azure Data Factory の継続的インテグレーションとデプロイに関するベスト プラクティスのいくつかについて説明します。 |

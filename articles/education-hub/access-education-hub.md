# <a name="accessing-the-azure-education-hub"></a>Azure Education Hub にアクセスする

Azure Education Hub にアクセスするには、学術助成が承認されたことを知らせる電子メール通知を既に受け取っている必要があります。 この電子メールには、承認されたクレジット額と [Education Hub](https://aka.ms/startedu) へのリンクが含まれています。 ほとんどのスポンサー アカデミック プランには、無料の Developer レベルの Azure サポートが含まれています。 セットアップ プロセスで問題が発生した場合、[Educator Services Desk](mailto:azuredu@microsoft.com) にお問い合わせいただくか、お住まいの地域の Microsoft の担当者にお問い合わせください。

> [!IMPORTANT]
> 学術助成を利用するには、Education Hub でクラスルームを作成し、対象の Azure クレジットにアクセスするクラスルーム内でサブスクリプションを使用する必要があります。

## <a name="how-to-access-the-azure-education-hub"></a>Azure Education Hub にアクセスする方法

1. ブラウザーで、Chrome の Incognito や Microsoft Edge の InPrivate など、プライベート ブラウジング タブを開きます。

    :::image type="content" source="media/access-education-hub/browsing-in-private.png" alt-text="プライベート ブラウザー ウィンドウを開きます。" border="false":::

1. Microsoft 資格情報を使用して [Azure portal](https://portal.azure.com) にサインインします。

   > [!IMPORTANT]
   > (機関の @domain.edu など) 組織または職場のアカウントでサインインしていることを確認します。 確認できたら、まず、ウィンドウの左側でこのオプションを選択します。 別のログイン画面が表示されます。

    :::image type="content" source="media/access-education-hub/sign-in.png" alt-text="組織のサインイン ダイアログ。" border="false":::

1. サインイン後、Azure portal に移動します。 Education Hub を見つけるには、 **[すべてのサービス]** に移動し、**Education** を検索します。
   初めてログインすると、[はじめに] ページが表示されます。

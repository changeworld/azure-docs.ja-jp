# <a name="what-is-the-azure-education-hub"></a>Azure Education Hub とは

Microsoft Azure [Education Hub](https://portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/quickstart) は、アカデミック ユーザーが多数の異なる Azure サブスクリプションにわたってクラウド クレジットをプロビジョニングおよび管理するのに役立つツールです。 これは、教職員メンバーが多数のクラウドベースの学生プロジェクトを管理する必要がある場合に便利です。 また、Education Hub は、Azure クレジットのニーズが不明なときに調査をする目的にも役立ちます。

ユーザーは、各サブスクリプション内または一括で、割り当てられたクレジットの額を簡単に調整できます。 また、サブスクリプションは、使用可能なクレジットを最大限に活用するために一時停止することもできます (たとえば、半期または四半期のプロジェクトが完了したとき)。

## <a name="prerequisites"></a>前提条件

Education Hub にアクセスするには、学術助成が承認されたことを示す電子メール通知を既に受け取っている必要があります。 この電子メールには、承認されたクレジット額と [Education Hub](https://aka.ms/startedu) へのリンクが含まれています。

学術助成を利用するには、Education Hub でクラスルームを作成し、対象の Azure クレジットにアクセスするクラスルーム内でサブスクリプションを使用する必要があります。

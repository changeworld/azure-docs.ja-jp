# <a name="alert-types-and-descriptions"></a>アラートの種類と説明

この記事では、Defender for IoT エンジンから生成される可能性のあるすべてのアラートの種類について説明します。 アラートは [アラート] ウィンドウに表示されます。ここで、アラート イベントを管理できます。 

## <a name="policy-engine-alerts"></a>ポリシー エンジンのアラート

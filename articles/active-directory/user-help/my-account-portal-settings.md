# <a name="view-or-change-your-settings-in-the-my-account-portal"></a>マイ アカウント ポータルで設定を表示または変更する

マイ アカウント ポータルでアカウントの設定 (言語やタイム ゾーンなど) を表示または変更するには、 **[マイ アカウント]** ポータルの **[Settings & Privacy]\(設定とプライバシー\)** ページを開きます。

## <a name="view-and-manage-your-language-and-regional-settings"></a>言語と地域の設定を表示および管理する

1. 職場または学校アカウントにサインインした後、、 https://myaccount.microsoft.com/ ページに移動します。

1. 左側のナビゲーション ウィンドウから **[Settings & Privacy]\(設定とプライバシー\)** を選択するか、 **[Settings & Privacy]\(設定とプライバシー\)** ブロックから **[View Settings and Privacy]\(設定とプライバシーの表示\)** リンクを選択します。

    ![[Settings & Privacy]\(設定とプライバシー\) リンクが強調表示されている [マイ アカウント] ページ](media/my-account-portal/my-account-portal-privacy.png)

1. *[言語と地域 (プレビュー)]* で、次の設定を確認または変更します。
    * *表示言語*:Microsoft Web サイトのボタン、メニュー、および一部のコンテンツに使用される言語。
    * *Preferred languages\(優先される言語\)* :コンテンツを作成または使用するときに理解して使用する言語。
    * *地域設定*:Microsoft Web サイトでは、この設定を使用して日付と時刻の書式が設定されます。
    * *タイム ゾーン*:Microsoft カレンダーで会議を効果的にスケジュールするために使用されます。

表示言語を変更した後、職場または学校のアカウントからサインアウトしてからサインインし、表示言語が更新されるようにすることをお勧めします。

> [!NOTE]
> 組織によっては、 *[表示言語]* 設定を管理できない場合があります。 次のスクリーンショットに示すように、表示言語が無効になっている場合は、IT 管理者に連絡してください。
>
> ![[マイ アカウント] ページで組織の [表示言語] 設定が無効になっている](media/my-account-portal/my-account-portal-managed-language-settings.png)

マイ アカウント ポータルの設定を変更した後、次のことができます。

- 接続されている[デバイス](my-account-portal-devices-page.md)を表示または管理する。

- [組織](my-account-portal-organizations-page.md)を表示または管理する。

- [サインイン アクティビティ](my-account-portal-sign-ins-page.md)を表示する。

- [言語と地域の形式の設定を現在使用している M365 Web アプリケーション](https://support.microsoft.com/office/change-your-display-language-and-time-zone-in-microsoft-365-for-business-6f238bff-5252-441e-b32b-655d5d85d15b)を表示する。

# <a name="micro-focus-devtest-environments-on-azure"></a>Azure における Micro Focus 開発/テスト環境

開発およびテストの環境を利用して Azure を使い始めることは、最新化の取り組みを推進するうえで有効な方法です。 IBM CICS (Customer Information Control System) や IMS (Information Management System) などのレガシ システムに依存しているメインフレーム アプリケーションは、パートナー ソリューションを使用して Azure 上でリホストできます。

Micro Focus は、実践的なメインフレーム移行とリホスト ツールを提供するために、マイクロソフトと提携しました。 Azure および Microsoft Windows プラットフォーム上で、開発、テスト、および運用のシナリオにわたって、メインフレーム アプリケーションを実行できます。

Micro Focus では、既存の COBOL と PL/I の業務アプリケーションを Azure に拡張する開発リホストとテスト リホストのソリューションを提供しています。

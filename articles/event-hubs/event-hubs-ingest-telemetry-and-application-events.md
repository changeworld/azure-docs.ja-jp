# 大規模なテレメトリとアプリケーション イベントの取り込み
 
最新のアプリケーションは、実行中に大量の情報を作成します。それがモバイル アプリケーションのユーザーのアクションでも Web サイトのクリックストリームでも、このテレメトリ データは、長期的な分析、およびリアルタイムの稼動状況の監視や意思決定支援に役立ちます。Service Bus はこの情報を取得し、処理するのに必要なツールを提供します。

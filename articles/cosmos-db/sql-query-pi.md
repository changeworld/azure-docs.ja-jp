# <a name="pi-azure-cosmos-db"></a>PI (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

 PI の定数値を返します。  
  
## <a name="syntax"></a>構文
  
```sql
PI ()  
```  
   
## <a name="return-types"></a>戻り値の型
  
  数値式を返します。  
  
## <a name="examples"></a>例
  
  次の例では、`PI` の値を返します。  
  
```sql
SELECT PI() AS pi 
```  
  
 結果セットは次のようになります。  
  
```json
[{"pi": 3.1415926535897931}]  
```

# <a name="azure-synapse-analytics-business-intelligence-partners"></a>Azure Synapse Analytics のビジネス インテリジェンス パートナー

Azure Synapse 内で Synapse SQL プールを使用して完全なデータ ウェアハウス ソリューションを作成するため、業界をリードする幅広いツールから選択できます。  この記事では、Azure SQL Data Warehouse をサポートする公式のビジネス インテリジェンス (BI) ソリューションを提供している Microsoft パートナー企業について説明しています。

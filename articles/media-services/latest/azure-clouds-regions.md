# <a name="azure-media-services-v3-clouds-and-regions-availability"></a>Azure Media Services v3 クラウドとリージョンの可用性

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

Azure Media Services v3 は、Azure Resource Manager から使用できます。 ただし、すべての Azure クラウドで Media Services のすべての機能を使用できるわけではありません。 このドキュメントでは、Media Services v3 の主要コンポーネントの使用可能性を示します。 次の表は、各リージョンで使用できる Media Services 機能を示しています。

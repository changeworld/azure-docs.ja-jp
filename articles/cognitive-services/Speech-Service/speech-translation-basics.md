# <a name="learn-the-basics-of-speech-translation"></a>音声翻訳の基本を学ぶ

Speech Service の中核となる機能の 1 つは、人間の音声を認識して他の言語に翻訳する機能です。 この記事では、ご利用のアプリと製品で Speech SDK を使用して、高品質の音声翻訳を実行する方法について説明します。

> [!TIP]
> クイックスタートのいずれかを完了する機会がなかった場合は、ご自分で動作方法の確認をして音声翻訳を試してみることをお勧めします。

> * [音声間翻訳](quickstarts/translate-speech-to-speech.md)

# <a name="quickstart-use-the-translator-text-api-to-detect-text-language"></a>クイック スタート:Translator Text API を使用してテキストの言語を検出する

このクイックスタートでは、Translator Text REST API を使用して、指定されたテキストの言語を検出する方法について説明します。

#  Microsoft Azure アプリ向けの Application Insights

*Application Insights はプレビュー段階です。*


課題を検出し、問題を解決して、アプリケーションを継続的に改善します。実際のアプリケーションの問題を今すぐ診断する。ユーザーがそれで何を行っているかを理解します。

構成は非常に簡単であり、数分以内で結果を確認できます。

# <a name="azure-synapse-analytics-data-management-partners"></a>Azure Synapse Analytics データ管理パートナー
Azure Synapse の SQL Analytics 機能を使用してデータ ウェアハウス ソリューションを作成するため、業界をリードする幅広いツールから選択できます。 この記事は、Azure Synapse をサポートするデータ管理ツールおよびソリューションを提供している Microsoft パートナー企業について説明しています。

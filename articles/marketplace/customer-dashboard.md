# <a name="customers-dashboard-in-commercial-marketplace-analytics"></a>コマーシャル マーケットプレース分析での [顧客] ダッシュボード

この記事では、パートナー センターの [顧客] ダッシュボードについて説明します。 このダッシュボードには、成長傾向などの顧客に関する情報が、グラフィカルかつダウンロード可能な形式で表示されます。

パートナー センターの [顧客] ダッシュボードにアクセスするには、**コマーシャル マーケットプレース** で、 **[[分析]](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)**  >  **[顧客]** を選択します。

>[!NOTE]
> 分析の用語の詳細な定義については、「[コマーシャル マーケットプレース分析の用語とよく寄せられる質問](./analytics-faq.md)」を参照してください。

## <a name="customers-dashboard"></a>[顧客] ダッシュボード

[顧客] ダッシュボードには、プランを購入した顧客のデータが表示されます。 次の項目のグラフィカル表示を見ることができます。

- アクティブな顧客およびチャーン顧客の傾向
- 既存の顧客、新規の顧客、およびチャーン顧客を含む顧客成長傾向
- 注文別および使用量別の顧客
- 顧客のパーセンタイル 
- 注文数と使用量ごとの顧客の種類
- 地域別顧客数
- 顧客詳細テーブル
- 顧客ページ フィルター

> [!NOTE]
> 顧客の獲得からパートナー センターでの報告までの最大待ち時間は、48 時間です。

## <a name="elements-of-the-customers-dashboard"></a>[顧客] ダッシュボードの要素

次のセクションでは、[顧客] ダッシュボードの使用方法とデータの読み方について説明します。

### <a name="month-range"></a>月範囲

各ページの右上隅には、月範囲の選択が表示されます。 過去 6 か月または 12 か月の月範囲を選択することで、あるいは 12 か月を最大期間とするカスタムの月範囲を選択することで、 **[顧客]** ページのグラフの出力をカスタマイズします。 既定の月範囲 (計算期間) は 6 か月です。

:::image type="content" source="./media/customer-dashboard/month-range-filters.png" alt-text="[顧客] ページの月フィルターの画像。":::

> [!NOTE]
> 視覚化ウィジェットおよびエクスポート レポートのすべてのメトリックでは、ユーザーが選択した計算期間が優先されます。

### <a name="active-and-churned-customers-trend"></a>アクティブな顧客およびチャーン顧客の傾向

このセクションでは、選択した計算期間の顧客成長傾向を確認します。 メトリックと増加傾向は折れ線グラフで表され、グラフの線にマウス ポインターを合わせると、各月の値が表示されます。 ウィジェットの **アクティブな顧客** の下にあるパーセント値は、選択した計算期間中の増加量を表します。 たとえば、次のスクリーンショットは、選択した計算期間に 0.92% の増加を示しています。

[![[顧客] ページの [顧客] ウィジェットの画像。](./media/customer-dashboard/customers-widget.png)](./media/customer-dashboard/customers-widget.png#lightbox)

_顧客の種類_ には、新規、既存、チャーンの 3 つがあります。

- 新規顧客は、選択した月のうちに初めて 1 つ以上オファーを購入した顧客です。
- 既存顧客は、選択した月よりも前に 1 つ以上オファーを購入したことがある顧客です。
- チャーン顧客は、以前に購入したすべてのオファーをキャンセルした顧客です。 チャーン顧客は傾向ウィジェットの負の軸で表されます。

### <a name="customer-growth-trend-including-existing-new-and-churned-customers"></a>既存の顧客、新規の顧客、およびチャーン顧客を含む顧客成長傾向

このセクションには、新規、既存、チャーンを含むすべての顧客の傾向と数が、月ごとの増加傾向と共に表示されます。

- 折れ線グラフは、顧客全体の増加率を表します。
- 月の列は、新規、既存、チャーンの各顧客を積み重ねた顧客数を表します。
- チャーン顧客数は X 軸の負の方向に表示されます。
- 特定の凡例項目を選択して、より詳細なビューを表示できます。 たとえば、新規顧客のみを表示するには、凡例から新規顧客を選択します。
- グラフ内の列にカーソルを合わせると、その月の詳細のみが表示されます。

[![[顧客] ページの [顧客の傾向] ウィジェットの画像。](./media/customer-dashboard/customers-trend.png)](./media/customer-dashboard/customers-trend.png#lightbox)

### <a name="customers-by-ordersusage"></a>注文/使用量別顧客数

**注文/使用量別顧客数** グラフには、[注文]、[正規化した使用]、および [生の使用] の 3 つのタブがあります。 **[注文]** タブを選択して、注文の詳細を表示します。

:::image type="content" source="./media/customer-dashboard/customers-by-orders-usage.png" alt-text="[顧客] ページの [注文別および使用量別の顧客] ウィジェットの [注文] タブの画像。":::

次のことを考慮してください。

- リーダー ボードには、注文数によって順位付けされた顧客の詳細が表示されます。 顧客を選択すると、隣接した [詳細]、[SKU 別の注文]、[シート別の SKU] の各セクションに、顧客の詳細が表示されます。
- 顧客プロファイルの詳細は、発行者が所有者ロールを使用してサインインしている場合、この領域に表示されます。 パブリッシャーが共同作成者ロールを使用してサインインしている場合、このセクションの詳細は使用できません。
- **[SKU 別の注文]** ドーナツ グラフには、プランに対して購入された注文の内訳が表示されます。 注文数が最も多い上位 5 プランが表示され、残りの注文は **[残りすべて]** にまとめられます。
- **[シート別の SKU]** ドーナツ グラフには、プランに対して注文されたシートの内訳が表示されます。 シート数が最も多い上位 5 プランが表示され、残りの注文は **[残りすべて]** にまとめられます。

**[正規化した使用]** または **[生の使用]** タブを選択して、使用状況の詳細を表示することもできます。

- リーダー ボードには、使用時間によって順位付けされた顧客の詳細が表示されます。 顧客を選択すると、隣接した [詳細]、[オファー別の正規化した使用]、および [仮想マシン (VM) サイズ別の正規化した使用] の各セクションに顧客の詳細が表示されます。
- 顧客プロファイルの詳細は、発行者が所有者ロールを使用してログインしている場合、この領域に表示されます。 パブリッシャーが共同作成者ロールを使用してログインしている場合、このセクションの詳細は使用できません。
- **[オファー別の正規化した使用]** のドーナツ グラフには、オファー別の使用量の内訳が表示されます。 使用量が最も多い上位 5 プランが表示され、残りのオファーは **[残りすべて]** にまとめられます。
- **[VM サイズ別の正規化した使用]** のドーナツ グラフには、さまざまな VM サイズ別の使用量の内訳が表示されます。 正規化された使用が最も多い上位 5 VM サイズが表示され、残りの使用量は **[残りすべて]** にまとめられます。

### <a name="top-customers-percentile"></a>上位顧客のパーセンタイル

**[上位顧客のパーセンタイル]** グラフには、[注文]、[正規化した使用]、[生の使用] の 3 つのタブがあります。 _[上位顧客のパーセンタイル]_ は注文数によって決定され、X 軸に沿って表示されます。 Y 軸には顧客の注文数が表示されます。 第 2 の Y 軸 (折れ線グラフ) には、合計注文数に対する累積の割合が表示されます。 折れ線グラフ上の点にポインターを合わせると、詳細を表示できます。

[![[顧客] ページの [上位顧客のパーセンタイル] ウィジェットの [注文] タブの画像。](./media/customer-dashboard/top-customer-percentile.png)](./media/customer-dashboard/top-customer-percentile.png#lightbox)

### <a name="customer-type-by-orders-and-usage"></a>注文数と使用量ごとの顧客の種類

**[Orders/usage by customer type]\(顧客タイプごとの注文数/使用量\)** グラフには、注文数、正規化された使用時間、および生の使用時間が、新しい顧客と既存の顧客とを分けて、グラフで [orders]\(注文\)、[normalized usage]\(正規化された使用\)、[raw usage]\(未調整の使用\) のいずれを選択したかに応じて表示されます。

このグラフには次のものが示されています。

- 新規顧客は、同じ月 (Y 軸) のうちに初めて 1 つ以上オファーを購入したか、使用を報告した顧客です。
- 既存顧客は、レポート内の月 (Y 軸) よりも前にプランを購入したか、使用を報告したことがある顧客です。

[![[顧客] ページの [顧客の種類別注文数] ウィジェットの [注文] タブの画像。](./media/customer-dashboard/orders-by-customer-type.png)](./media/customer-dashboard/orders-by-customer-type.png#lightbox)

### <a name="customers-by-geography"></a>地域別顧客数

このヒートマップには、選択した計算期間での顧客の合計数と、地域ディメンションに対して新たに追加された顧客の割合が表示されます。 マップ上の明るい色は顧客数が少ないことを表し、暗い色は顧客数が多いことを表します。 テーブル内のレコードを選択すると、1 つの国または地域が拡大されます。

[![[顧客] ページの [地域別注文数] ウィジェットの [注文] タブの画像。](./media/customer-dashboard/customers-by-geography.png)](./media/customer-dashboard/customers-by-geography.png#lightbox)

次のことを考慮してください。

- マップを動かして正確な場所を見ることができます。
- 特定の場所にズーム インできます。
- ヒートマップには、特定の場所での顧客数、注文数、正規化された使用量の詳細を表示するための補助グリッドがあります。
- グリッドで国またはリージョンを検索して選択すると、マップ内の場所にズームできます。 地図内の **[ホーム]** ボタンを選択すると、元のビューに戻ります。

### <a name="customer-details-table"></a>顧客詳細テーブル

**顧客詳細** テーブルには、上位 1,000 顧客の番号付きリストが、プランの 1 つを最初に購入した日付で並べ替えて表示されます。 [詳細] リボンの展開アイコンを選択すると、セクションを展開できます。

次のことを考慮してください。

- 顧客の個人情報は顧客が同意した場合にのみ参照できます。 この情報は、所有者ロール レベルのアクセス許可でサインインしている場合にのみ表示できます。
- グリッドの各列で並べ替えることができます。
- レコード数が 1,000 件未満の場合は、.CSV または .TSV ファイルにデータを抽出できます。
- レコード数が 1,000 を超える場合、エクスポートされたデータは今後 30 日間、ダウンロード ページに非同期で配置されます。
- 関心のあるデータのみを表示するには、テーブルにフィルターを適用します。 データは、会社名、顧客 ID、Marketplace サブスクリプション ID、Azure ライセンスの種類、購入日、失効日、顧客の電子メール、顧客の国、地域、都道府県、市区町村、郵便番号、顧客の言語などでフィルター処理できます。
- 保護された顧客によってオファーが購入された場合、 **[Customer Detailed Data]\(顧客詳細データ\)** 内の情報はマスクされます (************)。
- Customer ディメンションの詳細 (会社名、顧客名、顧客の電子メールなど) は、Azure Marketplace や Microsoft AppSource トランザクションのレベルではなく、組織 ID のレベルにあります。

### <a name="customers-page-filters"></a>顧客ページ フィルター

[顧客] ページのフィルターは、[顧客] ページ レベルで適用されます。 ユーザーは、複数のフィルターを選択して、選択した表示条件に合うグラフや、[詳細な注文データ] グリッド/エクスポートに表示したいデータを表示できます。 フィルターは、[顧客] ページの右上隅で選択した月範囲に対して抽出されたデータに適用されます。

> [!TIP]
> ウィジェットの右上隅にあるダウンロード アイコンを使用して、データをダウンロードできます。 "サムズアップ" アイコンまたは "サムズダウン" アイコンをクリックすることで、各ウィジェットに関するフィードバックを提供できます。

# <a name="texconv---texture-conversion-tool"></a>TexConv - テクスチャ変換ツール

TexConv ツールは、バッチ処理されたテクスチャ変換を実行するための外部コマンドライン ツールです。
このツールは、次の git リポジトリにあります。[TexConv ツールの GitHub リポジトリ](https://github.com/microsoft/DirectXTex/wiki/Texconv)。

Remote Rendering の場合は、変換手順によって共通のイメージ ファイル形式を処理して、テクスチャを適切なランタイム形式に変換できるため、テクスチャを変換する必要はありません。 ただし、このツールは、テクスチャをカスタム設定でローカルに変換する場合には役立ちます。
